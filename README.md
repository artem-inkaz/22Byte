# Приложение для просмотра новостей 
## Используется FREE API https://newsapi.org/
## Новости привязаны к Locale по умолчанию
## Стек: Многомодульная архитектура, MVVM, Room, Navigation Components,Coroutines, Hilt, WebView, FirebaseCrashlitycs, Timber
